HOW TO USE:
-----------

FROM WEB INTERFACE:
1. Start server: python start_web_interface.py
2. Open http://localhost:5000
3. Fill in policy parameters
4. Click "Generate Prediction"
5. Scroll to "Export Results" section
6. Click any export button to download

FROM PYTHON CODE:
```python
from export_utils import export_manager

# Export to PDF
pdf_buffer = export_manager.export_to_pdf(result_data, "Policy Name")

# Export to Excel
excel_buffer = export_manager.export_to_excel(result_data, "Policy Name")

# Export to CSV
csv_buffer = export_manager.export_to_csv(result_data)

# Export to PowerPoint
ppt_buffer = export_manager.export_to_powerpoint(result_data, "Policy Name")
```
